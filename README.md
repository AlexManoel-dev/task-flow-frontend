# Taskflow (Angular Port)

Port do seu projeto Next.js para Angular 18 (standalone) com Tailwind, preservando identidade visual
via `app/globals.css` original.

## Rodar localmente
```bash
npm i
npm start
```

## Rotas
- `/` (Landing)
- `/login`, `/register`, `/forgot-password`
- `/dashboard`
- `/dashboard/projects/:id`
- `/admin`
